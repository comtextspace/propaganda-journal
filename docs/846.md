# В одном из районов Ровенской области продают школу

**2009-04-06** Автор неизвестен

Районный совет Млиновского района Ровенской области принял решение о выставлении на продажу недостроенной школы в селе Ставрив. Инициативу по продаже школы проявил глава районного совета и начальник отделения образования района. Жители села поражены цинизмом решения - оно было принято несмотря на то, что сельское собрание выступило против продажи школы.

В селе практически в каждом дворе есть дети, а четыре жительницы села - матери-героини. Теперь, когда школу продадут, детям придется ходить несколько километров в соседние села. Самое печальное то, что глава села поддержал предложение о продаже школы, хотя ее достройка была одним из главных пунктов его предвыборной программы.
