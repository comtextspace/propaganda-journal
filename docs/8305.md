# В Украине с 1 мая правительство повысит тарифы  на газ и на коммунальные услуги для населения

**2014-03-23** Автор неизвестен

"До среды миссия МВФ продолжит работать, мы обсуждаем условия. Первое повышение тарифов для населения будет не ранее 1 мая, потому что технологически это раньше сделать невозможно", - отматил министр энергетики и угольной промышленности Украины Юрий Продан.
