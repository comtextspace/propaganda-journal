# Японцы планируют создать лифт в космос

**2014-09-24** 

Японская корпорация "[Obayashi](http://www.obayashi.co.jp/english/about/)" планирует создать[ космический лифт](http://www.abc.net.au/news/2014-09-21/japanese-construction-giants-promise-space-elevator-by-2050/5756206) к 2050 году. Лифт здесь имеется в виду в прямом смысле: конструкция будет представлять собой кабину, которая будет перемещаться по тросу, один конец которого будет закреплен на океанской платформе, а другой - на МКС. В качестве троса японцы собираются использовать [углеродные нанотрубки](https://ru.wikipedia.org/wiki/%D0%A3%D0%B3%D0%BB%D0%B5%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B5_%D0%BD%D0%B0%D0%BD%D0%BE%D1%82%D1%80%D1%83%D0%B1%D0%BA%D0%B8), которые в сто раз прочнее стали. Другие имеющиеся материалы, из которых можно сделать трос, не выдержат даже своего собственного веса. На сегодняшний день, длина имеющихся нанотрубок - около 3 см. Для реализации проекта нужен трос длинной 96 000 км. Испытания планируют начать в 2030 году, [говорит](http://www.abc.net.au/news/2014-09-21/japanese-construction-giants-promise-space-elevator-by-2050/5756206) Йодзи Исикава, менеджер по исследованиям и разработкам в "Обаяси"

Революционность проекта в том, что такой лифт позволит доставлять грузы в космос намного дешевле, чем сейчас (сегодня доставка 1 кг груза стоит 22 000 долларов). Но основной целью проекта является развитие космического туризма. Предполагается, что лифт будет способен доставлять 7 человек на расстояние в 96 тыс. км. Такое путешествие займет 7 дней.
