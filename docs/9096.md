# Японцы планируют создать лифт в космос

**2014-09-24** 

Японская корпорация "Obayashi" планирует создать космический лифт к 2050 году. Лифт здесь имеется в виду в прямом смысле: конструкция будет представлять собой кабину, которая будет перемещаться по тросу, один конец которого будет закреплен на океанской платформе, а другой  - на МКС. В качестве троса японцы собираются использовать углеродные нанотрубки, которые в сто раз прочнее стали. Другие имеющиеся материалы, из которых можно сделать трос, не выдержат даже своего собственного веса. На сегодняшний день, длина имеющихся нанотрубок  - около 3 см. Для реализации проекта нужен трос длинной 96 000 км. Испытания планируют начать в 2030 году, говорит Йодзи Исикава, менеджер по исследованиям и разработкам в "Обаяси"

Революционность проекта в том, что такой лифт позволит доставлять грузы в космос намного дешевле, чем сейчас (сегодня доставка 1 кг груза  стоит 22 000 долларов). Но основной целью проекта является развитие космического туризма. Предполагается, что лифт будет способен доставлять 7 человек на расстояние в 96 тыс. км. Такое путешествие займет 7 дней.
