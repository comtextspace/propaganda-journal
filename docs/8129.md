# СБ ООН соберется в ночь на вторник для обсуждения ситуации в Украине

**2014-03-03** Автор неизвестен

Совет Безопасности ООН в 15:30 по местному времени (22:30 киевское время) проведет заседание для обсуждения украинского вопроса, сообщают в понедельник западные СМИ.

В сообщениях говорится, что заседание созвано по инициативе России, которая планирует изложить собственное видение событий в Украине.
