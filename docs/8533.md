# В Донецке прошел митинг в поддержку референдума

**2014-04-27** С. Р.

Площадь Ленина в Донецке стала традиционным «местом паломничества» сторонников «Донецкой Народной Республики». 27 апреля здесь прошел митинг в поддержку референдума, который состоится 11 мая.

Напомним, что на голосование будет выноситься только один вопрос: поддерживают ли жители региона независимость ДНР.

По данному вопросу на митинге выступил представитель временного правительства республики Денис Пушилин. Примечательно, что бывший функционер МММ прямо заявил о своих симпатиях к Советскому Союзу.
