# Почти вся электронная почта - спам

**2009-04-09** 

"Майкрософт" провела исследование, в ходе которого выяснилось, что 97% электронных сообщений, отправляемых через интернет, являются спамом. Спам - один из основных каналов передачи вирусов. Специалисты компании подсчитали, что на 10 000 компьютеров приходится 86 зараженных вирусами. По их оценкам, некулонно растет количество зараженных файлов. Э. Гибсон, советник Microsoft по вопросам компьютерной безопасности, объясняет популярность спама среди хакеров тем, что гораздо легче получить доступ к компьютеру, используя слабости пользователя, чем штурмуя изощренные средства защиты.

97%! Они достойны стать в один ряд с такими фактами современности, как расходы на косметику, которых хватило бы на пищу для миллионов голодающих и гостиницы для собак! И все это - для вящего блага частной инициативы по торговле всем на свете.
