# Первые два трупа на улице Грушевского

**2014-01-22** 

Один из погибших еще вчера упал с коллонады сталиона им. Лобановского, откуда боевики забрасывали милицию бутылками с зажигательной смесью и обстреливали фейерверками. Согласно сообщениям, умер он еще вчера днем, а сообщения об этом появились только ближе к сегодняшнему утру. Притом, СМИ говорили, что смерть подтвердили организаторы майдана, но, получается, сами о ней не сообщали - даже в [милицию](http://www.rbc.ua/rus/news/accidents/mvd-proveryaet-informatsiyu-o-smerti-cheloveka-v-rezultate-22012014092500)

Тело другой жертвы найдено сегодня утром будто бы с огнестрельными ранениями. По [данным](http://euromaidan.rbc.ua/rus/ustanovlena-lichnost-pogibshego-na-ul-grushevskogo---istochnik-22012014092900) РБК убытый - один из охранников майдана, житель Днепропетровской области, судя по его [страничке](http://vk.com/id135274323) ВК, армянский националист по убеждениям, Сергей Нигоян. 5 канал распространяет [версию](http://www.youtube.com/watch?feature=player_embedded&v=nVORfDfw1BY#t=16), согласно которой Беркут зашел в тыл бойцам майдана и оттуда атаковал этого человека. По другой [версии](http://comments.ua/politics/447856-grushevskogo-ognestrelnogo-raneniya.html), его убили снайперы.

В МВД заявили, что огнестрельного оружия на вооружении их сотрудников на Грушевского нет.

В любом случае, нужно ждать эскалации событий. Оппозиция начала формировать так называемые отряды самообороны из числа футбольных фанатов.

О.Л.
