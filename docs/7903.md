# Первые два трупа на улице Грушевского

**2014-01-22** 

Один из погибших еще вчера упал с коллонады сталиона им. Лобановского, откуда боевики забрасывали милицию бутылками с зажигательной смесью и обстреливали фейерверками. Согласно сообщениям, умер он еще вчера днем, а сообщения об этом появились только ближе к сегодняшнему утру. Притом, СМИ говорили, что смерть подтвердили организаторы майдана, но, получается, сами о ней не сообщали - даже в милицию

Тело другой жертвы найдено сегодня утром будто бы с огнестрельными ранениями. По данным РБК убытый - один из охранников майдана, житель Днепропетровской области, судя по его страничке ВК, армянский националист по убеждениям, Сергей Нигоян. 5 канал распространяет версию, согласно которой Беркут зашел в тыл бойцам майдана и оттуда атаковал этого человека. По другой версии, его убили снайперы.

В МВД заявили, что огнестрельного оружия на вооружении их сотрудников на Грушевского нет.

В любом случае, нужно ждать эскалации событий. Оппозиция начала формировать так называемые отряды самообороны из числа футбольных фанатов.
