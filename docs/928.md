# Бессрочная забастовка

**2009-04-26** Автор неизвестен

В городе Шостка смская обл. с 21 апреля идёт забастовка протеста на заводе «Звезда». Рабочие требуют выплатить долги по зарплате с августа прошлого года, что в сумме составляет почти 7 млн грн.

По сообщению Gazeta.ua, рабочие собираются бастовать, пока им не выплатят долги. 10 дней назад был назначен новый директор Иван Козлов, фигурант нескольких уголовных дел. Рабочие считают его виновным в разорении завода.

Казенный завод "Звезда" - единственный пороховой завод на Украине по утилизации боеприпасов.

*По материалам rabkor.ru, Gazeta.ua*
