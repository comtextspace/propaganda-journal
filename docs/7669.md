# Долги по зарплатам в Украине бьют новые рекорды

**2013-11-16** К.М.

Долги по заработной плате в Украине приближаются к рекордному максимуму. По данным Государственной службы статистики, задолженность достигла более миллиарда гривен, сообщает 5 канал.

Половина из них — это долги предприятий, которые финансируются из государственного или местных бюджетов. В правительстве же такую ??статистику объясняют влиянием проблем в частном бизнесе.

Между тем, ситуация на некоторых предприятиях становится напряженной, профсоюзы обещают вывести людей на улицы, чтобы добиться выплаты заработанного.
