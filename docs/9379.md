# Махмуд Аббас подтвердил намерение о создании независимой Палестины со столицей в Иерусалиме

**2015-01-13** Автор неизвестен

Об этом он [заявил](http://russian.people.com.cn/n/2015/0113/c31520-8835493.html) во время визита в Турцию. Президент Турции Р. Эрдоган отметил, что Турция поддерживает усилия палестинцев по созданию независимого палестинского государства.

На данный момент уже 135 государств-членов ООН признают независимость Палестины.
