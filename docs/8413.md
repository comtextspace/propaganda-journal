# Митингующие захватили здание ОГА в Харькове

**2014-04-06** Автор неизвестен

Как сообщает сайт [Status quo](http://www.sq.com.ua/rus/news/obschestvo/06.04.2014/v_harkove_shturmuyut_zdanie_obladministracii/) около девяти часов вечера пророссийски настроенные активисты начали штурм здания областной администрации. Впрочем, милиция особого сопротивления не оказывала. Через десять минут все желающие могли войти в здание, над которым был водружен российский флаг.
