# В центре Киева появится часовня памяти героев «Небесной сотни»

**2014-03-24** 

На улице Институтской в центре Киева появится часовня памяти героев Небесной сотни.

Как передает корреспондент агентства "Интерфакс-Украина", часовня  расположится между гостиницей "Украина" и верхним выходом из станции  метро "Крещатик".

На место уже привезли деревянные бревна, из которых по принципу  деревянного сруба будет построена часовня, а также крест и купол.

По словам строителей, материалы для часовни привезены из Закарпатья, а сам объект будет сдан в течении месяца.

Ранее на этом месте находилось несколько палаток участников Майдана,  которые во время штурма силовиками 18 февраля были полностью сожжены.  Сейчас здесь построен импровизированный мемориал из брусчатки, на  который водружена мемориальная плита в память погибших. Позже рядом был  установлен деревянный крест.

А вот с расследованием того, по чьей же вине образовалась эта "Небесная сотня", новопровозглашенные власти не спешат, для которых складывающаяся ситуация вполне устраивает. А массам рефлексировать по поводу произошедшего и вовсе необязательно. Лучше поставить свечку.
