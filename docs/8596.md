# Пентагон предоставил снаряжение украинским пограничникам

**2014-05-07** 

Министерство обороны США предоставило погранслужбе Украины финансирование на закупку специального снаряжения для охраны границы, сообщает Пентагон.

«Посольство в Киеве закупило и передало топливные насосы, проволочные заграждения, аккумуляторы и запчасти для автомобилей, бинокли и средства связи для украинских пограничников, чтобы они могли использовать это для наблюдения и обеспечения безопасности границы», – сказал представитель Пентагона Стив Уоррен, передает РИА Новости.

По его словам, оборудование было закуплено непосредственно в Украине «в последнее время».

Помощь США Киеву сейчас делится на две категории: предназначенную для пограничной службы и для регулярных армейских частей. Помощь для армии продолжает поступать по соответствующим каналам, сказал представитель Пентагона.
