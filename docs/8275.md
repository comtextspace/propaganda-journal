# Правительство Украины утвердило план мероприятий по эвакуации украинских граждан из Крыма

**2014-03-19** 

Правительство Украины утвердило план мероприятий по эвакуации украинских граждан из Крыма и их размещению на территории материковой Украины.

Об этом сообщил министр Кабинета министров Украины Остап Семерак.

"Сегодня принят соответствующий план мероприятий. Решение принято правительством и Советом национальной безопасности и обороны. План предусматривает поручение всем органам центральной исполнительной власти провести подготовительный этап и выполнить задачи, которые стоят сегодня", - сказал он на пресс-конференции в среду в Киеве.

Министр Кабмина выразил надежду, что этот процесс не будет угрожающим, но украинские власти должна подготовиться к "надлежащей реакции на возможные миграционные движения".

Кроме этого, по словам О.Семерака, правительство ведет диалог с органами местной власти, которые совместно с Министерством социальной политики Украины размещают крымчан.
