# Митинг возле резиденции Ахметова (обновляется)

**2014-05-25** С. Р.

**16:45** Митингующие расходятся. Штурма резиденции не было. Временное правительство ведет переговоры с Р. Ахметовым.

**15:30** Ополченцы ведут переговоры с личной охраной Рината Ахметова, чтобы она предоставила возможность беспрепятственно пройти внутрь. По некоторым данным, самого олигарха в резиденции нет.

**14:25** В Донецке пять тысяч человек - противники президентских выборов - собрались у резиденции бизнесмена Рината Ахметова, одного из самых богатых людей Украины. Они требуют, чтобы он покинул Донбасс. Звучат лозунги "Донецкая республика", "Требуем экскурсии", "Ахметов - враг народа".
