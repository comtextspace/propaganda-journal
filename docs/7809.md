# Великобритания введет языковой тест для мигрантов, которые хотят получать социальные выплаты

**2013-12-24** Г. М.

![Великобритания введет языковой тест для мигрантов, которые хотят получать социальные выплаты](http://img.gazeta.ru/files3/845/5786845/sssss-pic4_zoom-1000x1000-23721.jpg)

Всех трудовых мигрантов Великобритании, претендующих на получение социальных выплат, обяжут сдавать тест на знание английского языка. Для того, чтобы получить социальные выплаты им предстоит продемонстрировать владение «стандартом английского языка». Это означает, что им придется ответить на специально разработанные вопросы. Только справившись с индивидуально-адаптированным заданием, мигранты смогут подать заявление на оформление соцпособий. Политики объясняют это ужесточение требований желанием защитить британскую систему льгот, выплачивая помощи только тем, кто действительно может работать и в полной мере участвовать в жизни общества. Не исключен вариант, что от мигрантов будут требовать предоставить также информацию о том, какие меры они сделали для самоутверждения на территории государств, о жилье и семейном положении, сообщает Daily Mail.

Хотя со слов политиков требование ужесточения условий для мигрантов звучит как забота о британской системе льгот, это больше напоминает попытку в очередной раз сэкономить на социальных выплатах рабочим.
