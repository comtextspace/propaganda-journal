# Депутаты «Донецкой Народной Республики» отказались переносить референдум о суверенитете региона c 11 мая

**2014-05-08** 

В ДНР не подддержали предложение, озвученное вчера В.Путиным. Решение было принято на собрании депутатов в Донецке 8 мая единогласно. За проголосовали 78 депутатов. Об этом сообщил один из лидеров ДНР Андрей  Пургин. «Это не наше решение, это решение народа Донбасса. Народ Донбасса  получил шанс совершить подвиг, и мы не имеем права лишать его этого  шанса», - заявил он.
