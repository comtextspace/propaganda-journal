# Из-за долгов Берлускони не будет праздновать Рождество

**2013-12-23** В. М.

Из-за гигантских долгов бывший премьер-министр Италии Сильвио Берлускони отменил традиционное празднование Рождества на принадлежащей ему вилле Бельведере недалеко от Милана, где раньше к празднику свозили моделей [и несовершеннолетних проституток](/7017.html) со всей Италии устанавливали гигантскую ель, о чем пишет Corriere della Sera.

Вся прислуга на вилле, где тот жил со своей бывшей женой, уже получила открытки с рождественскими поздравлениями письма об увольнении, и с 23 декабря двери виллы будут закрыты.

Соответствующее решение принято из соображений экономии, которой теперь вынужден придерживаться политик. После громких разоблачительных скандалов, после которых Берлускони стал популярнее в западной прессе, по всей видимости, финансовые махинации уже не проходят для него так удачно, как раньше.
