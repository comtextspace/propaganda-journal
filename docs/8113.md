# Новая власть планирует создать комиссию для ведения переговоров с РФ

**2014-03-02** Автор неизвестен

Представители новой власти в Киеве намерены сформировать специальную комиссию и обратиться к России с просьбой о переговорах, сообщил в воскресенье лидер украинской партии "УДАР" Виталий Кличко.
