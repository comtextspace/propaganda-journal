# Цена на газ для Украины повысилась

**2014-04-01** Автор неизвестен

Как заявил глава "Газпрома" Алексей Миллер, цена на газ для Украины с 1 апреля вырастет до 385,5 долларов за тысячу кубометров. То есть, она выросла на 117 долларов по сравнению с 268,5 долларов за тысячу кубов, которые Украина платила в первом квартале. Точнее, украинская сторона в первом кваратале ничего не платила, что Миллер назвал в качестве одной из причин повышения цены.

На данный момент, по его словам, задолженность "Нафтогаза" составляет 1,7 млрд долларов.

Кстати, кабинет министров Украины при пересмотре бюджета страны на текущий год заложил новую цену на газ с "потолком" в 480 долларов за тысячу кубов, а устно Яценюк вообще говорил о 500 долларах.

Вполне возможно, что российская сторона прислушается к пожеланиям украинских руководителей. Ведь это будет уже не первый случай, когда в результате настойчивых действий украинского правительства цена на газ для Украины повышается. Так премьер Тимошенко в свое время в результате искуссно проведенных переговоров сумела повысить ее сразу на 53% по сравнению с первоначально предложенной «Газпромом».
