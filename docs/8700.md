# Выборы в Луганской области проходят без нарушений

**2014-05-25** 

Мало того, они отлично проходят даже без избирательных комиссий и избирателей. Как сообщил сайт РБК,  в Луганской области по состоянию на 12:00, по словам и.о. председателя Луганской облгосадминистрации Ирины Веригиной проголосовало 18% избирателей.

Это при том, что по сообщению того сайта РБК сегодня были открыты только 2 из 12 окружных избирательных комиссий.

Для сравнения, в Днепропетровской обл., где работают все 100% комиссий, по состоянию на 12:00 проголосовал 21% избирателей.

А может, избирательные комиссии вообще не нужны, для того, чтобы выборы были признаны вполне демократическими? А в том, что всевидящие западные наблюдатели их признают таковыми, можно не сомневаться.
