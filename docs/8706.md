# В центре Мариуполя идет бой

**2014-05-26** Автор неизвестен

В Мариуполе началась атака на захваченное неделю назад вооруженными сторонниками ДНР здание на ул. Греческой. Слышна стрельба из автоматического оружия. По свидетельствам местных жителей, атакуют люди в черной униформе. Ходят слухи, что перед атакой они применили газы.
