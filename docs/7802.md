# Покончил с собой известный борец с депрессией

**2013-12-22** В.М.

В Нью-Йорке покончил с собой знаменитый автор, писавший о борьбе с депрессией. 32-летний Нед Визини спрыгнул с крыши родительского дома в Бруклине, как бы подтверждая эффективность собственных методов.



Известность американцу принес асфальт, проложенный возле дома автобиографический роман «Эта забавная история», экранизированный в 2010-том. Герой — депрессивный подросток, который в итоге осознает — жизнь прекрасна и прыгает с крыши родительского дома.
