# «Беркут» за считанные минуты разогнал митингующих из-под Администрации президента

**2013-12-01** 

Штурм Администрации президента прекратился. Его сейчас разогнали. «Беркут» начал наступать и бить митингующих, которые бросились бежать. Есть раненые. Наступление правоохранителей длилось считанные минуты.
