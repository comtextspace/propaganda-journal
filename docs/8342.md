# Януковича исключили из Партии регионов

**2014-03-28** 

Сегодняшний съезд Партии регионов, как передает Интерфакс-Украина, постановил исключить Виктора Януковича из рядов партии.



Борис Колесников официально подтвердил это решение. Кроме Януковича из регионалов также исключили: Николая Азарова, Александра Клименко, Сергея Арбузова, Андрея Клюева и Эдуарда Ставицкого.



По словам Колесникова, они удовлетворили просьбу президента, несмотря на то, что она была передана через СМИ.
