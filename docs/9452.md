# Россия является главным врагом Америки

**2015-02-17** 

Согласно опросу, проведенному компанией Gallup 8-11 февраля 2015 года, Россия заняла первое место в «рейтинге враждебности» по отношению к Соединенным Штатам. Злейшим врагом своей страны ее считают 18% американцев. На втором месте с 15% - КНДР. Китай считают врагом Америки 12% респондентов, Иран - 9%.

Некоторые украинские СМИ для надежности увеличили эту цифру до 49%. Видимо, для большей объективности.

Еще три года назад врагом Америки Россию считали только 2% респондентов.

Кстати, российские граждане отвечают Штатам полной «взаимностью» - 64 % опрошенных еще прошлым летом считали Соединенные Штаты главным врагом России.
