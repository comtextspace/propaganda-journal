# Эштон посетит Украину с визитом 4-5 марта

**2014-03-03** 

Верховный представитель ЕС по иностранным делам Кэтрин Эштон приедет в Киев с визитом во вторник, 4 марта.

Как сообщил директор департамента информационной политики МИД Украины  Евгений Перебийнис, визит будут двухдневным и завершится 5 марта.

Ранее К.Эштон планировала прибыть в Киев 3 марта.
