# Украинские войска обстреляли центр Донецка

**2014-07-29** С. Р.

Сегодня подверглись обстрелу жилые кварталы в центре Донецка. Войска вели огонь со стороны аэропорта. Снаряды попали в два дома по ул. Розы Люксембург.

Осколки разбили все окна в подъезде и повредили стоящие рядом гаражи. В ходе обстрела погиб один мужчина, число раненых пока неизвестно.

Приблизительно в том районе находится штаб «ополченцев». Судя по всему, артиллерия целилась именно туда.

За сегодня это уже второй обстрел жилых кварталов Донецка. В 5:00 снаряды разворотили несколько домов в частном секторе по ул. Стратонавтов.
