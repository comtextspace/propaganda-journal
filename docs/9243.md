# Агитационный автопоезд «Классовая борьба вместо мировой войны»

**2014-11-11** 

Продолжается движение антивоенного автопоезда "Klassenkampf statt Weltkrieg" по маршруту Франкфурт-Регенсбург. Решение о проведении второй за этот год агтационной поездки было принято в связи с активизацией немецкой милитаристской пропаганды. В этот раз маршрут проходит по территории Западной Германии. Движение автопоезда началось 8 ноября. Окончание движения запланировно на 23 ноября. Одной из целей второй поездки и связанных с ней мероприятий являеся охват антивоенной агитацией рабочих промышленных райнов Западной Германии. Маршрут был выбран исходя из намерения охватить местности, где расположены крупные предприятия, владельцы которых наживаются на военных заказах.
