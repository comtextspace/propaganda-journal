# Команда хакеров ФБР будет отслеживать террористов?

**2013-12-09** Искатель

«Теперь ФБР будут следить за пользователями Интернета с помощью их электронной почты.

По данным американской газеты The Washington Post, работники ФБР создали специальный вирус для того, чтобы с помощью интернета вести слежку за подозреваемыми террористами.

Профессиональная команда хакеров, специально созданная ФБР, разработала вирусную программу, которая поможет отслеживать преступников. Подозреваемый террорист и его действия будут фиксироваться каждый раз, когда он будет заходить в учетную запись электронной почты с любой точки мира.

Самое мощное программное обеспечение наблюдения ФБР может тайно скачивать файлы, фотографии и сохраненные сообщения электронной почты, или даже получать изображения в реальном времени путем активации камер, подключенных к компьютерам.

Критики сравнивают данную технику с физическим поиском, предусматривающим сбор широкого спектра информации, которая, часто, не имеет прямого отношения к какому-либо преступлению.

Стоит отметить, что в прошлом году инспекторам федерального ведомства удалось с помощью вирусной программы разыскать человека, который несколько раз угрожал организацией взрывов в американских университетах».

Интересно, а как же факт, что компьютерные вирусы распространяются по сети неконтролируемо? Наверное, первая фраза этой новости является ключевой.
