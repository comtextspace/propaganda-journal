# «Свобода» не проходит в Верховную Раду

**2014-10-28** 

По результатам обработки 83% бюллетеней «Свобода» не добирает треть поцента для преодоления прохождного барьера. Как сообщает радио «Свобода», ее результат составляет 4,69% при необходимых 5%.

Правда, несколько членов этой партии, судя по предварительным результатам, проходят по одномандатным округам. Это Андрей Ильенко, который набрал 39,55% голосов в 215 округе города Киева, Юрий Левченко - округ 223 в Киеве, Александр Марченко в округе № 90 Киевской области.

Интересно, что на Западной Украине ни один свободовец не прошел, а пресловутая Ирина Фарион в одном из округов Львовской области оказалась аж на третьем месте.

В Верховной Раде также будут два представителя «Правого сектора». Д. Ярош избран в Днепропетровской области, Б. Береза - в Киеве.
