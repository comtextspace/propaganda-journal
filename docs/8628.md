# Предварительные результаты явки на референдум в ДНР

**2014-05-11** С. Р.

Явка избирателей на референдуме за независимость «Донецкой народной республики» к 20.00 составляет:

1) по всей территории ДНР - 70%;

2) Донецк - 73%, самая высокая явка в Калининском р-не - 83%;

3) Славянск - 50%;

4) Красноармейск - 79%, голосование пока приостановлено из-за захвата горадминистрации батальоном «Днепр».
