# Сгенерированная на компьютере «девочка» Sweetie привлекла 1000 педофилов

**2013-11-06** Mel

Голландские активисты из организации [Terre des Hommes](http://www.terredeshommes.org/) завершили [эксперимент](http://www.terredeshommes.org/wp-content/uploads/2013/11/Webcam-child-sex-tourism-terre-des-hommes-NL-nov-2013.pdf) по идентификации педофилов в чат-комнатах. Для этого они применили метод ловли «на живца», причем в качестве приманки использовали компьютерную модель 10-летней филиппинской девочки под ником Sweetie. Непосредственно в чате работали активисты Terre des Hommes.

![undefined](http://www.xakep.ru/post/61545/sweetie.jpg)

По оценке ООН, в каждый момент времени в онлайне находится около 750 тыс. педофилов, при этом только в Филиппинах их «обслуживают» через интернет десятки тысяч детей, возрастом от шести лет. К сожалению, проблема педофилии усугубляется с каждым годом.

![undefined](http://www.xakep.ru/post/61545/chat.jpg)

*Одна из чат-сессий*

За десять недель операции на контакт со Sweetie в разных чатах вышло более 20 000 мужчин, из них около 1000 человек из 71 страны предложили ей деньги. Большинство преступников — из богатых западных стран: 254 из США, 110 из Великобритании. На третьем месте Индия (103 человека). Что характерно, не удалось обнаружить ни одного пользователя из России.

На видео показано, как работает Sweetie. Для анимации 3D-модели в реальном времени использовали живого актера.

4 ноября 2013 года эксперимент Terre des Hommes был закончен, досье на всех идентифицированных мужчин передали в Интерпол.
