# Необычная акция протеста: металлурги ликвидирующегося завода переплавили выпущенную продукцию

**2009-04-26** 

К необычной форме протеста прибегли металлурги французского завода "Ренкаст", расположенного на востоке страны у города Тонон-ле-Бен (департамент Верхняя Савойя). В знак протеста против намеченного закрытия их предприятия они переплавили уже готовую продукцию, предназначенную для автомобильных компаний "Рено" и "Пежо-Ситроен", сообщает ИТАР-ТАСС.



По сведениям из профсоюзных источников, в печь были отправлены многие десятки уже выплавленных алюминиевых компонентов машин. В результате полностью была уничтожена продукция, предназначенная для сборки моделей "Твинго" на заводах "Рено" и 80 % отливок для различных моделей "Пежо-Ситроен".



Профсоюзы подчеркнули, что таким образом металлурги намерены выразить свое недовольство ходом переговоров с руководством предприятия, касающихся тех условий, на которых они будут уволены при закрытии завода.



В настоящее время по решению коммерческого суда компания "Ренкаст", специализировавшаяся на литых алюминиевых компонентах для автомобилей, полностью ликвидируется. Суд разрешил работу компании под юридической опекой только до середины июня.



На заводе, открытом в 1963 году, работает более 180 человек. Всего на пяти заводах "Ренкаст" занято 850 человек.
