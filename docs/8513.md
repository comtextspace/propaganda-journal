# В Международном уголовном суде открыто предварительное расследование событий в Украине

**2014-04-25** Автор неизвестен

Прокурор Международного уголовного суда (МУС) открыла предварительное расследование по фактам преступлений, совершенных до и во время событий, сопровождавших уход президента Украины Виктора Януковича от власти.

"Прокурор МУС Фату Бенсуда решила открыть предварительное расследование ситуации в Украине для выяснения, существуют ли основания для начала полного расследования", - говорится в пресс-релизе МУС, распространенном в пятницу.
