# Украина отозвала посла из Армении

**2014-03-21** 

Украина отозвала посла в Армении Ивана Кухту для политических консультаций из-за позиции Армении касательно ситуации в Крыму.

Об этом на брифинге сообщил директор департамента информационной политики МИДа Евгений Перебийнис, пишут [Українські Новини](http://un.ua/).

«Сегодня принято решение отозвать посла Украины в Армении Ивана Кухту для политических консультаций», - сказал Перебийнис.

Напомним, президент Армении Серж Саргсян считает референдум о статусе Крыма, который состоялся 16 марта, примером реализации прав народов на самоопределение.
