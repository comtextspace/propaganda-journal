# Китайские власти не пустили британскую парламентскую делегацию в Гонконг

**2014-12-03** Автор неизвестен

Как [сообщает](http://russian.news.cn/china/2014-12/03/c_133829134.htm) агентство Синьхуа, правительство Китая отказало выдаче виз членам комитета по иностранным делам Палаты общин парламента Великобритании, которые собирались приехать в охваченный волнениями Гонконг «для сбора информации по демократическому процессу в Китае».

Согласно этому сообщению, китайские власти предупредили о том, что дальнейшие настойчивые попытки Великобритании обеспечить въезд своей парламентской делегации в Специальный административный район Сянган (так официально называют в китае Бывшую британскую колонию Гонконг) будут фактически сводиться к тому, чтобы "подняв камень, ударить им по своим ногам".

По всей видимости, китайцы решительно настроены не допустить развития ситуации по типу киевского Майдана, когда многочисленные западные политические деятели беспрепятсвенно участвовали в антиправительственных выступлениях.

Надо полагать, китайцы еще не состоянии понять, что колониальный статус для Гонконга - это и есть истинная демократия с точки зрения «международных стандартов», а настоящий суверинитет и самостоятельность - это когда правительство состоит из иностранцев.
