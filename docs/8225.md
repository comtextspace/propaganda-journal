# Верховная Рада распустила парламент Крыма

**2014-03-15** Автор неизвестен

Верховная Рада поддержала постановление за досрочное прекращение полномочий Верховной Рады АР Крым.

Соответствующее постановление поддержали 278 нардепов.

Из них от Партии регионов - 19, Родина - 83, УДАР - 37, "Экономическое развитие" - 30, "Суверенная европейская Украина" - 30, Свобода - 35, КПУ - 0, внефракционные - 44.

Однако сомнительно, чтобы это решение как-то изменило ситуацию, связанную с завтрашним референдумом.
