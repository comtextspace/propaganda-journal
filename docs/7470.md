# Папа Римский призывает священников изучать цифровые технологии

**2013-09-22** М.Г.

Папа Римский Франциск призвал священников, монашествующих и мирян активно изучать цифровые технологии, чтобы эффективно использовать их в проповеди христианства современному миру, сообщает РИА Новости.

Папа Римский в своем обращении к участникам пленарной ассамблеи Папского совета по социальной коммуникации в субботу отметил, что "именно сейчас, в эпоху глобализации, растет дезориентация, одиночество, неопределенность в отношении смысла жизни".

Понтифик напомнил о старом правиле паломников, о котором говорил святой Игнатий: "Тот, кто сопровождает паломников и идет вместе с ними, должен придерживаться их шага, то есть не отставать и не опережать".

Активное изучение цифровых технологий христианскими организациями с целью проповедования христианства, является лишь одним из способом поддержания влияния этой религии в обществе.

В современном мире человек действительно не уверен в завтрашнем дне, подвержен стрессам. Но для решения подобных проблем необходимо рассмотрение экономических условий их возникновения для изменения этих условий.
