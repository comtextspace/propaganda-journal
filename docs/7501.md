# В Китае раскрыта преступная сеть торговцев людьми

**2013-09-30** Автор неизвестен

Правоохранительные органы Китая раскрыли преступную сеть торговцев людьми. По делу проходит 301 человек, которые подозреваются в связях с преступными группами.

В операции по разоблачению огромной преступной группировки, занимающейся торговлей людьми, принимали участие представители правоохранительных органов из 11 провинций КНР. В частности преступники похищали и продавали детей из провинций Юнань и Сычуань.

Похищения детей очень распространены в Китае и считаются одной из самых серьезных проблем страны. Политика "одна семья - один ребенок" и тот факт, что китайцы традиционно отдают предпочтение сыновьям, привели к увеличению случаев похищения детей и женщин в последние годы.
