# Сторонники «Донецкой народной республики» захватили горсовет Дебальцево

**2014-05-06** 

Дебальцево - крупнейший железнодорожный узел недалеко от Горловки. Противники киевской власти заняли горсовет, не встретив никакого сопротивления. По их требованию состоялось заседание городского совета, на котором депутаты поддержали решение о проведении 11 мая областного референдума, на который вынесен только один вопрос: "Согласны ли вы с созданием "Донецкой народной республики"?

Напоминаем, что вчера, 5 мая, решение о проведении областного референдума о статусе региона [принял](http://www.oblrada.lg.ua/content/zayavlenie-prezidiuma-luganskogo-oblastnogo-soveta-3) президиум Луганского областного совета.
