# Владимир Путин во вторник выступит с посланием Федеральному собранию по Крыму

**2014-03-17** 

Президент РФ Владимир Путин выступит во вторник с внеочередным посланием Федеральному собранию РФ, которое будет посвящено ситуации вокруг Крыма. В этот же день Госдума планирует принять заявление по Крыму.

Выступление президента с посланием по Крыму намечено на 15 часов, сообщает Интерфакс. Пленарное заседание палаты во вторник будет проходить с 10 утра до обеда. После этого депутаты отправятся в Георгиевский зал Кремля, где президент обнародует свое послание Федеральному собранию.

Во вторник Госдума планирует принять заявление по ситуации в Крыму, отмечает РИА Новости. «Завтра утром будет заявление по Крыму по предложению лидеров четырех фракций и Сергея Евгеньевича Нарышкина», - сообщил первый вице-спикер нижней палаты Иван Мельников. Известно, что заявление будет рассматриваться палатой в самом начале пленарного заседания, то есть до обращения президента.

Скорее всего, речь идет о парадном мероприятии, которое в общем-то, напрашивается - не каждый день и даже год происходит приращение территории, тем более бескровное.

Поэтому и говорится об обращении по Крыму. Вопросы Восточной Украины, видимо, если будут затронуты, то сугубо опосредовано.
