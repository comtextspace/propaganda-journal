# Кличко пригрозил Турчинову переизбранием

**2014-03-25** 

Лидер УДАРа Виталий Кличко пожаловался на то, что новая власть принимает решения без консультаций со всеми участниками коалиции и сказал, что если такое будет продолжаться, может встать вопрос о переизбрании головы Верховной Рады.

Поводом послужило голосование по заявлению об отставке и.о. министра обороны Тенюха.

Неделю назад написать заявление об отставке предложил Турчинову А. Гриценко.
