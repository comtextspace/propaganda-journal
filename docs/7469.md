# В Донецке прошла выставка картин Алексея Полякова

**2013-09-22** С. Р.

![В Донецке прошла выставка картин Алексея Полякова](http://news.sevas.ru/upload/images/2011/11/v_sevastopole_startoval_vtoroj_mezhdunarodnyj_plenehr_akvamarin_ii/v_sevastopole_startoval_vtoroj_mezhdunarodnyj_plenehr_akvamarin_ii-2.jpg)

С 28 августа по 21 сентября в Донецком областном художественном музее по случаю 70-летия со дня рождения прошла выставка Народного художника Украины Алексея Ивановича Полякова. Всего насчитывалось около 80 полотен мастера, работающего преимущественно в области станковой живописи.
