# Названы семь самых красивых городов мира

**2014-12-09** 

По [сообщению](http://www.prensalatina.ru/index.php?option=com_content&view=article&id=90121:7&opcion=pl-ver-noticia&catid=2&Itemid=106) агентства «Пренса Латина» столица Кубы попала в этот список. Кроме в нем фигурируют города Ла-Пас (Боливия), Бейрут (Ливан), Доха (Катар), Дурбан (Южная Африка), Куала-Лумпур (Малайзия) и Виган (Филиппины).

Кампания New7 Wonders Cities началась в 2011 году. Рассматривались 200 000 городов-кандидатов из 220 разных стран, - заявил президент фонда организации Бернард Вебер, обнародовав результаты.

В октябре прошлого года жюри, которое состояло из гуруппы экспертов и представителей общественности, отобрало 28 городов, потом из них - 21, затем - 14 и, наконец, выделили семь, которые и получили титул «чудо-городов».
