# В Китае полиция освободила 92 похищенных ребенка

**2013-09-28** 

Злоумышленники похищали детей в провинциях Юньнань и Сычуань на юго-западе страны и затем переправляли их по всей стране для продажи.

Китайская полиция провела крупную операцию по освобождению похищенных детей, сообщили власти. В результате были вызволены 92 ребенка и арестовано более 300 человек, подозреваемых в торговле детьми.

Злоумышленники, предположительно, похищали детей в провинциях Юньнань и Сычуань на юго-западе страны и затем переправляли их по всей стране для продажи.

По сообщению корреспондентов, в последнее время торговля детьми стала для Китая серьезной проблемой. Основной причиной этого называется политика «одна семья - один ребенок».
