# В Южной Корее схвачен сержант, срелявший по своим сослуживцам

**2014-06-24** Автор неизвестен

Газета Женьминьжибао со ссылкой на корейское агентстсво Ренхап [сообщило](http://russian.people.com.cn/n/2014/0624/c31520-8745484.html) об аресте 22-летнего сержанта по фамилии Лим, убившего накануне пять и ранившего семь военнослужащих армии Южной Кореи. Вчера он был схвачен живым после попытки покончить жизнь самоубийством.

Инцидент произошел 21 июня вечером в демилитаризованной зоне, разъединяющей РК и КНДР. В 8:15 по местному времени сержант Лим открыл огонь по своим сослуживцам, а потом скрылся.
