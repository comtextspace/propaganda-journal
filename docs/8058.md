# Вместо «беркута» придут другие - «с другой философией»

**2014-02-26** Автор неизвестен

В Украине Арсен Аваков основал новое подразделение МВД вместо спецподразделения "Беркут", которое [было ликвидировано сегодня](/8049.md). Об этом сегодня, 26 февраля, заявил и.о.министра внутренних дел. "Я расформировал спецподразделение "Беркут". Будет создана комиссия, которая будет исследовать деятельность этого подразделения. Виновники будут наказаны, а те, кто не принимал участие в расстрелах, получат места в новом министерстве после аттестации. Также я основал новое подразделение. Его функции и философия совсем другие", - сообщил он.

Тем временем в Севастополе создано муниципальное подразделение милиции "Беркут". Председатель Координационного совета по организации Севастопольского городского управления по обеспечению жизнедеятельности Севастополя Алексей Чалый подписал распоряжение "О создании специального муниципального подразделения милиции "Беркут". В нем, в частности, говорится, что подразделение создается "для поддержания в Севастополе общественного порядка, недопущения противоправных действий, провокаций, массовых беспорядков и мародерства".
