# ООН предлагает Украине повысить пенсионный возраст

**2009-04-16** Автор неизвестен

Украину с официальным визитом посетила представитель ООН, заместитель директора регионального бюро ПРООН для стран Европы и СНГ Джихан СУЛТАНОГЛУ. Данный визит совершен в рамках проекта Программы развития ООН в Украине. Представитель предоставила Украине новые рекомендации Организации Объединенных Наций по поводу дальнейшего развития Украины в условиях кризиса. Всего ООН подготовила около 170 рекомендаций для Украины, главной идеей которых является углубление рыночных отношений и развитие рыночных институтов. По мнению ООН, только таким образом можно справиться с кризисом. В частности Украине предложили повысить пенсионный возраст и уровнять возраст выхода на пенсию для мужчин и женщин, т. е. установить общий пенсионный возраст в 60 лет. Интерестно, знают ли представители ООН о том, какая в Украине на сегодняшний день продолжительность жизни? По официальным данным, средняя продолжительность жизни в Украине - 67,5 года. Это при том, что Украина занимает одно из последних мест по продолжительности жизни в Европе. Естественно, что с такой статистикой при повышении пенсионного возраста выплачивать пенсии уже будет некому, а многие до пенсии так и не доживут, ведь по статистике у около 40 % юношей 16-18 лет в Украине есть возможность не дожить до пенсии.
