# В Киеве проходит выставка работ детей с синдромом Дауна

**2014-03-22** К.М.

22–23 марта возле статуи Петра Сагайдачного в Киеве проходит выставка, посвященная Международному дню человека с синдромом Дауна.

Выставка состоит из художественных работ, нарисованных детьми из школы-интерната города Ровеньки.

«Воспитанники школы-интерната города Ровеньки через творческие работы хотят поделиться своим взглядом на мир», — описывают мероприятие организаторы.

Выставка проходит в при поддержке благотворительного фонда «Надія жити» («Надежда жить») с 10:00 до 18:00 с 22 по 23 марта.

Напомним, что международный день человека с синдромом Дауна отмечается 21 марта.
