# В Харькове арестован Константин Долгов

**2014-04-19** 

Главный редактор интернет-издания «[Глагол](http://glagol.in.ua/)», а также активист харьковского антимайдана Константин Долгов несколько часов назад был задержан.

По некоторым данным, сейчас он находится в Киевском районном суде города Харькова (ул. Блюхера, 7).

Сейчас там собирается стихийный митинг.
