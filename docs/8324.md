# Газ для населения Украины подоражает в полтора раза

**2014-03-26** Mel

Цены на газ для населения Украины с 1 мая 2014 года вырастут на 50 процентов. Об этом в среду, 26 марта, сообщает агентство «УНИАН» со ссылкой на представителя «Нафтогаза Украины» Юрия Колбушина.

По его словам, также планируется на 40 процентов поднять тарифы на газ для предприятий теплокоммунэнерго, однако их эти меры коснутся чуть позже — с июля 2014 года. Окончательные показатели повышения тарифов будут отражены в правительственном постановлении с графиком повышения цен до 2018 года, который будет опубликован в ближайшее время.

На такой шаг власти Украины пошли по требованию Международного валютного фонда, с которым Киев ведет переговоры о выделении очередного транша финансовой помощи в 15-20 миллиардов долларов.

Так же значительное повышение тарифов обусловлено прекращением для Украины газовой скидки, обусловленной Харьковскими соглашениями от 2010 года. Она предусматривала скидку на газ для Украины в обмен на продление пребывания Черноморского флота РФ в Крыму. Потеряв Крым, Украина потеряла и газовые льготы.
