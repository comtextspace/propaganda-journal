# Коломойский предложил национализировать активы олигархов

**2014-07-14** Автор неизвестен

Конечно, не всех. В данном случае речь идет об активах Фирташа и Ахметова. Фамилии он называть не стал, а [назвал](http://top.rbc.ru/economics/14/07/2014/936372.shtml) только предприятия, которые принадлежат данным миллиардерам. Среди них областные газовые компании, "Укртелеком", "Захидэнерго", "Днепроэнерго", "Днепрооблэнерго".

При приватизации указанных активов условия прописывались "под одного покупателя", заявил Коломойский.

Насчет того, что делать с теми олигархами, которые недавно [выигрывали](/8258.html) тендеры, условия которых прописывались под одного продавца, Коломойский не сказал. Видимо, не увидел никакой связи между событиями.
