# Разработана система, превращающая пластик в съедобные грибы

**2015-01-02** Автор неизвестен

Австрийский дизайнер вместе с учеными из Утрехтского университета (Нидерланды) [разработала](http://www.wired.com/2014/12/mini-farm-produces-food-plastic-eating-mushrooms/) систему, которая позволяет превращать пластик в съедобные грибы. Речь идет о грибах *Pestalotiopsis Microspora*, которые способны разлагать полиуретан, не превращаясь при этом в токсичные отходы. Настольная система представляет собой фабрику для выращивания двух видов грибов: *Schizophyllum commune* и *Pleurotus ostreatus*, которые можно употреблять в пищу, несмотря на их довольно необычное пристрастие к пластику.

![undefined](http://www.popmech.ru/upload/iblock/2bf/2bf8cff9bd8a677955c1b924506076ff.jpg)

Для преобразования пластмассы в пищевой продукт ее помещают в камеру, где ультрафиолетовое излучение стерилизует материал и запускает процесс деградации. Затем пластик помещают в капсулу, изготовленную из агара, добавляют в нее грибницу и пластмасса превращается пушистую грибную массу. На данный момент такая переработка в грибнице занимает несколько месяцев, но исследователи работают над ускорением процесса за счет оптимизации условий роста.

![undefined](http://www.popmech.ru/upload/iblock/ee7/ee75d39841d841e673059d6d89b19d53.jpg)

С помощью пищевых добавок в агар можно выбрать практически любой вкус конечного продукта: сама Унгер уже опробовала «пластиковые» грибы и отметила, что они имеют «довольно нейтральный вкус». Впрочем, ученым еще предстоит убедиться в безопасности употребления данного продукта.
