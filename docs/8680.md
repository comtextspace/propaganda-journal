# Украина планирует «оздоровить экономику» за счёт войны на Донбассе

**2014-05-20** 

Минобороны и МВД планирует закупить две сотни БТР-4. Выпускать их будет Харьковское конструкторское бюро по машиностроению.

Как передаёт издание lb.ua, от выполнения этого заказа предприятиям будет достаточно средств для обеспечения выплаты заработной платы в течение года.

Заказ «позволит загрузить более 10 предприятий Харьковской области и смежных предприятий Украины оборонного комплекса», - заявил зампредседателя Харьковской ОГА Василий Хома.
